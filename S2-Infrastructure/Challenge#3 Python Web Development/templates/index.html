VoIP Server

3e Challenge groep 6

Wat is onze challenge?

Wij zijn challenge groep 6, dit is onze 3e challenge die we dit semester doen. In deze challenge gaan we onderzoek doen naar VoIP en het opzetten van een VoIP server. Onze challenge heeft omvat alle focusgebieden, deze zijn programming, connecting, provisioning, securing en managing.

Programming: Voor programming is er in python, m.b.v. flask deze webpagina gemaakt, hierop staan verschillende dingen zoals informatie over ons, informatie over de challenge en een beetje monitoring van de servers.
Connecting: Uiteraard is connecting ook een groot gedeelte van onze challenge, alle server moeten met elkaar kunnen praten en we moeten kunnen bellen via onze VoIP server.
Provisioning: Provisioning komt ook, een klein beetje, terug in het project, de server moet namelijk worden opgezet en ingesteld.
Securing: Alle communicatie moet veilig kunnen gebeuren.
Managing: Om de klantenservice in te richten wordt er ook een duidelijke en uitgebreide flowchart gemaakt, hierdoor hebben we een makkelijker beeld bij wat er gemaakt moet worden.

Deze webpagina is gemaakt door 3e challenge groep 6. Martijn Verlind, Jesper Hartsuiker, Daan de Weirdt en Luke Hendriks.
