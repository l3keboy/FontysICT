## Working Packet Sniffer!

Supported Protocols:
  - ICMP
  - UDP
  - TCP
    
Supported IP versions:
  - IPv4
  - IPv6
  
## Info
- Python3 Script
- To use this code make sure you download the latest version into your Linux (!!) machine.
- Make sure you have installed all the imports
